Works

Omnifood
My first proper website, for a fake app. Mostly HTML5 and CSS3, a little bit of jQuery. Responsive. Project for a Udemy course Build Responsive Real World Websites with HTML5 and CSS3.
Patatap Clone
Final project in the Front-End section of the Web Developer Bootcamp course on Udemy. Simplified clone of the website Patatap. Uses Paper.js framework to create shapes and Howler.js to play sounds.
To Do List
Simple, interactive, To Do List. Done using mostly jQuery. Part of the Web Developer Bootcamp course on Udemy.
Color Guessing Game
First "major" exercise using HTML, CSS and Vanilla JS. Simple color guessing game, in which a player has to guess which of the six squares has the color provided in the header in RGB code. Part of the Web Developer Bootcamp course on Udemy.
Purrfect
Simple "landing page" for a imaginary StartUp. Uses Bootstrap. Final exercise with only HTML and CSS from the Web Developer Bootcamp on Udemy.
